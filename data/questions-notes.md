# 题库编号参考

## 两种 ID 格式

- **手写题**：`chXX-NNN`（按章节编号）
- **模板生成**：`q0001~q0500`（全局编号）

新增题目用手写格式。

## 题目字段说明

| 字段 | 说明 |
|------|------|
| `id` | 唯一编号 |
| `type` | `single` 单选 / `multi` 多选 / `indefinite` 不定项选择 |
| `chapter` | 所属章节 |
| `stem` | 题干 |
| `options` | 4 个选项，A/B/C/D |
| `answers` | 正确答案 label，单选 1 个，多选可多个 |
| `explanation` | 答案解析 |
| `category` | `asset` 资产 / `liability` 负债 / `pl` 损益 / `other` 其他 |
| `material` | 可选，不定项选择题的案例材料 |

## 分类比例

资产(asset) : 负债(liability) : 损益(pl) : 其他(other) = 3 : 3 : 2 : 2
